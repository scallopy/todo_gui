# todo_gui

## Getting Started

1. Install Python3.8

    ```
    $ python --version
    Python 3.7.3
    $ which python3.8
    /usr/local/bin/python3.8
    $ alias python="/usr/local/bin/python3.8"
    $ python --version
    Python 3.8.4
    ```

2. Clone the repo: `$ git clone git@github.com:scallopy/todo_gui.git`
3. Create todo.txt and done.txt to store data
